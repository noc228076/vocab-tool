# 四六级背单词小工具 - 项目总结

## 项目概述

这是一个功能齐全的 Windows 系统托盘背单词工具，专门为中国大学生设计，支持 CET-4 和 CET-6 词汇学习。

## 核心功能

### 1. 系统托盘集成
- 最小化到系统托盘运行
- 右键菜单快速访问所有功能
- 不占用任务栏空间

### 2. 浮动单词窗口
- 可拖拽的浮动窗口
- 自动隐藏功能
- 支持自定义位置和显示时间

### 3. 学习模式
- **学习模式**：学习新单词
- **复习模式**：基于间隔重复算法的智能复习
- **测试模式**：英译中、中译英、听写测试

### 4. 间隔重复算法
- 基于 SM-2 算法
- 科学安排复习时间
- 自动调整复习频率

### 5. 学习统计
- 记录学习进度
- 统计正确率
- 追踪学习时间
- 连续学习天数

### 6. 单词本功能
- 收藏重要单词
- 随时复习收藏的单词
- 支持搜索功能

## 单词数据库

- **CET-4 词汇**：3739 个单词（含例句）
- **CET-6 词汇**：1615 个单词（含例句）
- **自定义词汇**：78 个单词
- **总计**：5432 个单词

单词包含：
- 单词拼写
- 音标
- 中文释义
- 例句
- 分类标签

## 技术实现

### 主要依赖
- `pystray`：系统托盘图标
- `Pillow`：图像处理
- `tkinter`：GUI 界面
- `sqlite3`：本地数据库

### 核心模块
1. **DatabaseManager**：管理学习进度和设置
2. **WordManager**：管理单词数据
3. **FloatingWordWindow**：浮动单词窗口
4. **VocabApp**：主应用程序

## 使用方法

### 快速开始
1. 双击 `启动背单词.bat`
2. 首次运行会自动安装依赖
3. 程序会最小化到系统托盘
4. 右键点击托盘图标开始学习

### 学习流程
1. **学习新单词**：选择「学习模式」
2. **复习单词**：选择「复习模式」
3. **测试自己**：选择「测试模式」
4. **收藏单词**：点击「收藏」按钮
5. **查看统计**：选择「学习统计」
