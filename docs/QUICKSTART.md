# 快速开始指南

## 第一步：安装 Python

确保你的电脑上安装了 Python 3.7 或更高版本。

下载地址：https://www.python.org/downloads/

安装时请勾选 "Add Python to PATH"。

## 第二步：运行程序

### 方式一：双击启动（推荐）

直接双击 `启动背单词.bat` 文件。

首次运行会自动：
1. 创建虚拟环境
2. 安装所需依赖
3. 启动程序

### 方式二：命令行启动

```bash
# 进入程序目录
cd vocab_tool

# 安装依赖
pip install -r requirements.txt

# 运行程序
python vocab_app.py
```

## 第三步：开始学习

1. 程序启动后会最小化到系统托盘（屏幕右下角）
2. 右键点击托盘图标，可以看到功能菜单
3. 选择「学习模式」开始学习新单词
4. 选择「复习模式」复习已学单词
5. 选择「测试模式」进行单词测试

## 功能说明

### 学习模式
- 学习新单词
- 显示单词、音标、释义、例句
- 支持「认识」「模糊」「不认识」三种评价

### 复习模式
- 基于间隔重复算法
- 自动安排复习时间
- 科学记忆，事半功倍

### 测试模式
- 英译中：看英文选中文
- 中译英：看中文选英文
- 听写模式：听发音写单词

### 单词本
- 收藏重要单词
- 随时复习
- 支持搜索

### 学习统计
- 记录学习进度
- 统计正确率
- 追踪学习时间
- 连续学习天数

## 常见问题

### Q: 程序无法启动？
A: 请检查是否已安装 Python，并确保已添加到 PATH。

### Q: 如何添加新单词？
A: 运行 `python add_words.py`，按提示输入单词信息。

### Q: 如何修改设置？
A: 右键点击托盘图标，选择「设置」。

### Q: 学习进度会丢失吗？
A: 不会，学习进度保存在本地数据库文件中。
