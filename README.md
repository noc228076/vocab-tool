# 四六级背单词

Windows 系统托盘背单词工具，后台静默运行，悬浮卡片随时学习。

## 快速开始

```bash
pip install -r requirements.txt
python run.py
```

或双击 `start.bat`（首次自动创建虚拟环境）。

**预编译版**：解压 `VocabTool_v1.0.zip`，直接运行 `VocabTool.exe`（无需 Python）。

## 功能

### 浮动卡片
- 自动弹词（定时弹出，最小间隔 5 秒）
- 顺序 / 乱序切换
- 词库切换（CET-4 / CET-6）
- 释义三档控制：单击手动显示 / 双击开启自动显示 / 再单击关闭
- 收藏单词、标记已认识
- 窗口可拖拽，位置自动记忆
- 透明度可调（设置面板 + 实时预览）

### 学习模式
- 四级词汇 / 六级词汇 / **混合模式**（各一半，随机打乱）
- 复习模式（SM-2 间隔重复算法）
- 单词评分：三档 — **认识**（等级+1）/ **模糊**（等级不变）/ **不认识**（等级-1）
- 快捷键：`Space` / `Enter` 显示释义，`1` 不认识，`2` 模糊，`3` 认识
- 学习完成后显示本次正确数

### 测试模式
- 英译中 / 中译英
- 四选一，即时反馈
- 快捷键：`1`-`4` 选择选项，`Space` / `Enter` 下一题

### 统计
- 已学单词数、已掌握（SM-2 等级 ≥4）、已认识（手动标记）
- 今日学习 / 复习 / 正确率 / 学习时间
- 累计学习时长、连续学习天数

### 导入单词书
支持两种格式，导入后自动去重归入 `custom` 分类：

**JSON** — 与 words.json 相同格式，或直接数组：
```json
[{"word":"hello","phonetic":"/həˈləʊ/","meaning":"int. 你好","example":"Hello, world!"}]
```

**TXT** — 每行一个词，支持音标和释义：
```
hello /həˈləʊ/ int. 你好
world 世界
# 注释行会被跳过
```

## 设置
- 弹词间隔（5–999 秒）
- 默认词库（CET-4 / CET-6）
- 显示模式（手动 / 自动）
- 每批学习数量
- 窗口透明度（0.30–1.00，滑块实时预览）

## 项目结构

```
vocab_tool/
├── src/vocab_app.py       # 主程序（~1400 行）
├── data/words.json        # 词库（5354 四六级词汇，全部含音标/例句）
├── tests/                 # 5 个测试文件（全部通过）
├── tools/                 # 13 个工具脚本
├── run.py                 # 启动器
├── start.bat              # Windows 启动脚本
├── requirements.txt       # 依赖：pystray, Pillow
├── VocabTool_v1.0.zip     # 预编译分发包
└── vocab_progress.db      # 学习进度（自动生成）
```

## 数据
- CET-4: 3739 词, CET-6: 1615 词, 合计 5354 词
- 所有单词均含音标（IPA）和例句
- 例句中目标单词橙色高亮，支持不规则动词词形变化

## 环境要求
- Windows 7/10/11
- Python 3.7+（源码运行）或 无需安装（预编译版）
